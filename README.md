Note Taking App

A simple and user-friendly note-taking application built with Flutter. This application allows users to create, view, update, and delete notes easily. Users can also attach images to notes, making it useful for storing reminders, tasks, and important information in one place.

Features
- Create new notes
- Edit existing notes
- Delete notes
- Attach images to notes
- Display note creation and modification dates
- Clean and modern user interface
- Floating Action Button for quick note creation
  
Screenshot
The application provides a simple note management interface where users can organize their notes efficiently.
Technologies Used
- Flutter
- Dart
- Material Design
- Local Storage (SharedPreferences / SQLite)
Project Structure

lib/
-> models/
-> screens/
-> widgets/
-> services/
-> main.dart

Getting Started
Prerequisites
- Flutter SDK
- Android Studio or VS Code
- Android Emulator or Physical Device
Installation
1. Clone this repository
git clone https://github.com/username/note_taking_app.git
2. Navigate to the project directory
cd note_taking_app
3. Install dependencies
flutter pub get
4. Run the application
flutter run
